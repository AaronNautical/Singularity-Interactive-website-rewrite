Web Application Developer

Singularity Interactive is looking for a Web Application Developer to join our team in creating world-class web applications for both internal products and external clients. Employees in this role will work with a team of software professionals to develop and maintain software and its related features with a focus on learning and gaining expertise. Candidates should be prepared to learn and adopt established patterns, best practices, and standards in order to contribute to one or more active projects.

Responsibilities:

  • Develop, design, implement, and maintain software projects and their individual features.
  • Coordinate and work with other developers, business and system analysts, and UI/UX designers.
  • Documents facets of the software project as a reference for further maintenance and upgrades.
  • Build and maintain web applications by contributing directly to front end and back end code.
  • Continually learn and improve web application development skills independently, via collaboration, and through mentorship.

Required Skills/Experience:

  • Has zero to two years of software development work experience.
  • Basic knowledge about IT from a software and hardware perspective, and computer programming basics.
  • Knows computer programming basics and its tools.
  • Knows at least one programming language with the ability to write basic code.
  • Has the ability to repair small, less significant bugs or errors in software, and work on internal features and tools of the software.
  • Git tools and collaboration workflows.
  • Basic to intermediate knowledge of JavaScript, TypeScript, and/or Go.
  • Basic to intermediate knowledge of HTML, CSS/SASS.

For more information or to apply for this position please contact: careers@singint.com

Unreal Engine Developer

Singularity Interactive is looking for a talented video game developer to help bring our game products to market. In this role, you will be responsible for working within our project's game engine (currently Unreal Engine 4), source code, and game assets to implement the game design, artistic direction, and overall experience based on input from stakeholders. Ideal candidates will be self-motivated, great communicators, and enjoy working in highly dynamic teams and projects.

Responsibilities:

  • Responsible for the development, design, implementation and maintenance of game software projects and their individual features.
  • Work closely with game directors, project managers, game designers, UI designers, game artists, animators, architects, and other developers to bring our game concepts to life.
  • Translate and implement our game designs into code and game assets with a level of polish that is competitive within our target markets.
  • Document every facet of the game software project as a reference for further maintenance and upgrades.
  • Working closely with product ownership to help prioritize features based on critical path and technical lift.
  • Mentor other software developers who are gaining proficiency with game technology.
  • Maintain up-to-date knowledge of Unreal Engine features, capabilities, best practices, and standards.

Required Skills/Experience:

  • Four or more years of game software development work experience in Unreal Engine, including contribution to shipped titles. Experience with additional game engines is a big plus.
  • Knowledgeable in C++ and Unreal Engine Blueprint with the ability to write code and create game assets which implement complex feature requirements.
  • Has the ability to troubleshoot and repair bugs or errors in game software.
  • Basic knowledge about IT from a software and hardware perspective, and computer programming basics.
  • Ability to operate in uncertainty and self-activate to gain clarity from teammates and stakeholders as necessary to accomplish assigned tasks.
  • Ability to work constructively within a team of varying opinions while leaning in with your expertise.

For more information or to apply for this position please contact: careers@singint.com

Web Software Reliability Engineer

Singularity Interactive is looking for an experienced and self-motivated Web Application Developer to join our team. For this position, an additional focus on DevOps support, and Cloud Infrastructure design and operations is required. The ideal candidate will be passionate about designing and maintaining infrastructure for scalable web applications while staying close to the supported application by also contributing to web application code.

Responsibilities:

  • Responsible for the development, design, implementation and maintenance of software projects and their individual features.
  • Coordinate and work with other developers, business and system analysts, and UX designers.
  • Documents every facet of the software project as a reference for further maintenance and upgrades.
  • Provides solutions based on previous experience.
  • Plan and execute the automation of all types of workloads in our cloud environments (primarily AWS)
  • Build and support serverless services as well as Kubernetes clusters in enterprise environments, leveraging and maintaining automation where appropriate
  • Manage CI tools, GitHub, Docker, and container orchestration tools
  • Create regular reports of costs and utilization associated with AWS services.
  • Work closely with other application developers to build out CI/CD pipelines and workflows
  • Contribute directly to the web application projects that you support
  • Creatively solve new complex problems while trying to keep the solution simple

Required Skills/Experience:

  • Minimum of five years experience in AWS and related cloud technologies
  • Minimum of three years experience contributing directly to web application client and/or server code
  • Two years of experience in the field of data center infrastructure management (UNIX/Docker/ Kubernetes/ Microservices)
  • Is competent in all or some stages of the systems development life cycle – analysis, design, development and testing, implementation, documentation, and evaluation
  • Must be well versed in generic administration tasks of managing Docker images, container networking and standard infrastructure maintenance tasks on Docker and Kubernetes platform
  • Experience writing automation scripts
  • Strong system administrator skills
  • Comfort with facilitating collaboration, open communication and reaching across functional border
  • Programming background in JavaScript/TypeScript, React, Node, and/or Go
  • In-depth experience with continuous integration and continuous deployment pipelines
  • Strong analytical and debugging skills
  • Ability to identify and articulate key trade-offs when making cloud and infrastructure decisions
  • Strong written and verbal communication skills
  • Must be a self-starter and motivated to work with people to get tasks accomplished
  • High level of customer responsiveness, excellent documentation and communication skills and attention to detail
  • Has basic skills in debugging software
  • Can demonstrate the initiative and method for solving different problems, and propose effective solutions for those problems.
  • Can set up a development environment on their own, and has the ability to mentor a team of junior developers.
  • Has the ability to perform tasks for several projects at the same time.
  • Can revise software code to make it more efficient and be able to detect bugs in the code.
  • Can create and write simple to moderately complex tests.

For more information or to apply for this position please contact: careers@singint.com
